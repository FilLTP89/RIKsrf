#Example files for testing the modeling codes
----------------------------------------------

####List of available examples:
 - `Napa`: Evaluates slip rate functions for the Napa earthquake. The example includes evaluation of apparent source time functions in subdirectory `apparentSTFs`
 and additional files to calculate full-wavefield seismograms using Axitra in subdirectory `1Dsynthetics`.
 - More will come.

Good luck!

####References
 - Galloviè, F. (2016). Modeling velocity recordings of the Mw6.0 South Napa, California, earthquake: unilateral event with weak high-frequency directivity, Seism. Res. Lett. 87, 2-14.
 - See also http://geo.mff.cuni.cz/~gallovic/.
